Play: Windows, Mac, and Linux Instructions

STEP 1: DOWNLOAD JAVA and install.

STEP 2: Download the installer for your platform. Note you might get a message saying this program "could be dangerous" suggesting you discard the program. Select to "Keep" it.

Mac download

STEP 3: Run the installer.

STEP 4: Run Strategic Initiative (by double-clicking on the icon on your desktop).

STEP 5: In "Account Settings," enter the same username and password you registered with on this site.

STEP 6: In the "Join Game" tab, select a game in the list of games and click "Join".

STEP 7: Then on the "Myu Games" tab, select a game on the list and click "Enter".

Common Problems

  • If you have a 64 bit jvm installed on your computer (e.g. you are a developer), then you can try the 64 bit Windows client or the 64 bit Linux client.
