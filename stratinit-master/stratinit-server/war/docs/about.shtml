About Us

Strategic Initiative is a free, Open-Source game created by me, Ken Stevens. I pay for the www.strategicinitiative.org game servers out of my own pocket. I'm currently hosting the servers at Rimu Hosting which I highly recommend to anyone looking for a server hosting. The cost for me to run the servers is $55.29 per month.

If I ever end up getting so many players that I need to increase my server capacity, then I'll run a short donation campaign to raise the funds to pay for the increased capacity. But for now, I'm happy to pay for the servers myself.

The full source code for Strategic Initiative is available on the stratinit project hosted by Google Code. Anyone is free to download the source code and host games on their own servers if they like. The code is licensed under the GNU General Public License v3, so if you make improvements to the game I'd like to know about them so I can incorporate your changes back into the project (if they're good changes :-).

A bit about me... Back in the 90's, I was one of the key coders on an old, text-based online strategy game called Empire which continues to live on to this day as Wolfpack Empire. Many of the ideas (supply, interdiction, interception...) in Strategic Initiative are drawn from this game. However Strategic Initiative is considerably simpler than Empire. Empire was the ultimate micro-management game.

I now work for a software consulting firm in Toronto, Canada, mostly as a Project Manager for software development projects. I don't get to code much at my day job, so for fun I write game software in my spare time.

I also create and publish board games, which you can check out over at Toccata Games.
