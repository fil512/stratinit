Windows, Mac, and Linux Instructions

In "Account Settings," enter the same username and password you registered with on this site.

In the "Join Game" tab, select a game in the list of games and click "Join".

Then on the "Myu Games" tab, select a game on the list and click "Enter".

Common Problems

  • If you are having trouble running the client, try right-clicking on "play" above and selecting "Save As..." and saving the stratinit-client-1.1.jnlp file somewhere. Then double-click on "stratinit-client-1.1.jnlp" to run the game.
  • Sometimes the client can get corrupted. You can re-install your client as follows.
    • Start -> Run... (Windows XP) or Start (Windows Vista / 7)
    • Type "javaws -viewer" in the box and press Enter
    • A window should come up with a list of installed programs
    • Delete all copies of "Strategic Initiative"
    • Close the window
    • Try downloading and running the client again
  • If you have a 64 bit jvm installed on your computer (e.g. you are a developer), then you can try the 64 bit Windows client or the 64 bit Linux client. Note that this client won't run from within a browser; you will need to download the jnlp file and then run it separately.
